# Dotfiles
```
Usage:
  dotfiles.sh
  dotfiles.sh {-I --install}   [-g] [-p]
  dotfiles.sh {-U --uninstall} [-g]
  dotfiles.sh {-T --test}
  dotfiles.sh {-h --help}

Options:
  -g  Configure GUI environment
  -p  Install system packages
```
