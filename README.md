#Dotfiles

##Setup
Run [setup.sh](setup.sh) to create symlinks to the configuration files in your home directory. Note that configuration will not performed for root users.

    git clone https://github.com/sneivandt/.dotfiles.git
    ./.dotfiles/setup.sh

##Software
* Bash
* Git
* i3
* Tmux
* urxvt
* Vim
* X Windows
